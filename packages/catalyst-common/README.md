A rust library providing functionality from dalek-cryptography along with some custom functions

## Install

Install Rust via the Rustup tool:

```curl https://sh.rustup.rs -sSf | sh```

If ```rustc --version``` fails, restart your console to ensure changes to ```PATH``` have taken effect.

## Testing

To build and run the tests:
```cargo test```

To test with key generation enabled:
```cargo test --features key-gen```

For benchmarking:
```cargo bench```
